Simple Linear Model

Simple linear regression function used to estimate specific values rather than classifications.

y = α + βx

where y is the estimate of the number of people occupying a location, x is the average number of devices connected to an AP from quarter past to quarter to the hour.

Multinominial Logistic Regression

Probablistic model where the probability is estimated that the outcome variable assumes a certain classification. Multinomial logistic regression used for the multi-class problem of several ground truth classifications.

k-Nearest Neighbour

This supervised learning algorithm is very intuitive. Suppose you want to identify which class the green circle belongs to. In this example, the circle belongs to one of two classes. This algorithm takes a vote from the circle’s nearest neighbours. For instance, if you set k to 3, it looks for the three closest occurrences and decides that the circle is a red triangle.

Support Vector Machine

Support vector machine is a supervised learning algorithm that treats each data point as a coordinate which are referred to as support vectors. The algorithm classifies each support vector by finding the hyper-plane that differentiates the classes. This means that the algorithm finds the frontier that best segregates the classes. This is done by maximizing the distances between nearest data point and hyper-plane. The goal is to maximize the margin.
